# CPU、GPU满血补丁
⚠️注意！⚠️加压有风险，请确保供电和散热足够
## 使用
```bash
sudo ./switch-dtb.sh
# 性能模式
./performance-governor.sh
# 平衡模式
./balance-governor.sh
```

## glmark2 score提升
注意: 在桌面或VT执行
```bash
./performance-governor.sh
glmark2 #or glmark2-drm for VT
# 分数应该在1025左右, 超之前是820上下
```
